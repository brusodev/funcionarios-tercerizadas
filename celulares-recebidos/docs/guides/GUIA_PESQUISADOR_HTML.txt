🎯 O QUE É?
─────────────────────────────────────────────────────────────────────────────

Um aplicativo WEB (HTML + JavaScript) totalmente funcional para pesquisar e 
visualizar dados de modelos de celulares a partir dos Mega JSONs gerados.

✨ CARACTERÍSTICAS:
─────────────────────────────────────────────────────────────────────────────

✓ Interface moderna e responsiva
✓ Busca em tempo real por modelo
✓ Filtro por ADM
✓ Visualização de dados por ADM específico
✓ Ranking dos 20 modelos mais comuns
✓ Barras de progresso mostrando percentual
✓ Design bonito com gradientes
✓ Funciona em desktop, tablet e mobile
✓ Sem necessidade de servidor - funciona localmente

📂 ARQUIVOS NECESSÁRIOS
─────────────────────────────────────────────────────────────────────────────

Para funcionar, o HTML precisa dos seguintes arquivos no MESMO DIRETÓRIO:

1. pesquisador_modelos.html ← O arquivo principal
2. MEGA_JSON_MODELOS_SIMPLIFICADO.json ← Dados dos modelos
3. MEGA_JSON_MODELOS_COMPLETO.json ← Dados completos com detalhes

Estrutura de pastas (exemplo):
```
celulares-recebidos/
├── pesquisador_modelos.html
├── MEGA_JSON_MODELOS_SIMPLIFICADO.json
└── MEGA_JSON_MODELOS_COMPLETO.json
```

🚀 COMO USAR
─────────────────────────────────────────────────────────────────────────────

OPÇÃO 1: Abrir no Navegador (Mais Fácil)
──────────────────────────────────────────
1. Localize o arquivo "pesquisador_modelos.html" no explorador
2. Clique duas vezes para abrir no navegador padrão
3. Pronto! O aplicativo carregará

OPÇÃO 2: Arrastar e Soltar
──────────────────────────
1. Abra seu navegador (Chrome, Firefox, Edge, etc)
2. Arraste o arquivo "pesquisador_modelos.html" para o navegador
3. O aplicativo abrirá automaticamente

OPÇÃO 3: Menu do Navegador
──────────────────────────
1. Abra seu navegador
2. Use Ctrl+O (ou Cmd+O no Mac)
3. Navegue até a pasta e selecione "pesquisador_modelos.html"
4. Clique em Abrir

💡 DICAS DE USO
─────────────────────────────────────────────────────────────────────────────

1. BUSCA RÁPIDA:
   - Não precisa digitar tudo - basta parte do modelo
   - Busca é case-insensitive (maiúscula ou minúscula)
   - Resultado atualiza conforme digita

2. FILTRO POR ADM:
   - Use quando buscar um modelo em ADM específico
   - Combina com a busca por texto
   - Deixe em branco para buscar em todos

3. TOP 20:
   - Ótimo para entender quais são os modelos mais comuns
   - As barras de progresso mostram importância
   - Clique na aba para atualizar a visualização

4. NAVEGAÇÃO:
   - Use as 3 abas para diferentes tipos de consulta
   - Os filtros só afetam a aba "Buscar Modelo"
   - Mude de aba quantas vezes quiser

❌ SOLUÇÃO DE PROBLEMAS
─────────────────────────────────────────────────────────────────────────────

PROBLEMA: "Erro ao carregar os arquivos JSON"
SOLUÇÃO:
  1. Certifique-se que os 3 arquivos estão no MESMO diretório
  2. Verifique os nomes dos arquivos (case-sensitive no Linux/Mac)
  3. Tente reabrir o HTML
  4. Limpe o cache do navegador (Ctrl+Shift+Del)

PROBLEMA: Interface muito lenta
SOLUÇÃO:
  1. Feche outras abas do navegador
  2. Tente em outro navegador
  3. Reinicie o navegador

PROBLEMA: Botões não funcionam
SOLUÇÃO:
  1. Ative JavaScript (verifique nas configurações)
  2. Tente recarregar a página (F5)
  3. Limpe cookies/cache
